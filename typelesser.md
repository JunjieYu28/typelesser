## 最终目标

构建一个 Windows 桌面 AI 语音听写工具，核心功能：

1. 全局热键唤醒：在任何应用中按下快捷键开始录音，松开结束
2. 语音转文字：调用 Whisper API 把录音转为原始文本
3. AI 润色：把原始转录文本用 LLM 润色成流畅自然的语言
4. 全局文字注入：把润色后的文字自动输入到当前焦点的输入框中，就像在键盘打字一样
5. 悬浮状态窗口：录音、处理中、完成各状态有视觉反馈
6. 系统托盘常驻：后台运行，不占用任务栏
